# 数字设计构成网站设计规划

## 设计风格

### 视觉风格

- **主题**：现代、专业、富有创意的教育平台
- **色彩方案**：
  - 主色：深蓝色（#1E40AF）- 代表专业性和学术性
  - 辅助色：橙色（#F59E0B）- 代表创造力和活力
  - 中性色：灰色系列用于文本和背景
- **字体**：
  - 标题：Noto Sans SC（思源黑体）- 现代、清晰
  - 正文：Inter / Noto Sans SC - 易读性强
- **布局风格**：卡片式设计，网格布局，响应式

### 设计原则

1. **清晰的信息层级**：使用卡片、标签、徽章等元素区分不同难度等级
2. **视觉引导**：通过色彩、图标、间距引导用户浏览
3. **模块化设计**：每个知识点、案例都是独立的卡片模块
4. **响应式布局**：适配桌面、平板、手机

## 网站结构

### 导航架构

```
首页 (Home)
├── 课程大纲 (Curriculum)
│   ├── Loop 1: 概念冲刺 (第1-2节)
│   ├── Loop 2: 形态冲刺 (第3-7节)
│   └── Loop 3: 精炼冲刺 (第8-12节)
├── 知识卡片 (Knowledge)
│   ├── 平面构成
│   ├── 色彩构成
│   └── 空间构成
├── 案例库 (Cases)
│   ├── 按专业分类
│   └── 按构成原理分类
├── AI工具箱 (AI Tools)
│   ├── 提示词模板
│   └── 工作流指南
└── 学习资源 (Resources)
    ├── 在线工具
    └── 拓展阅读
```

### 页面规划

#### 1. 首页 (Home)

- Hero区域：课程介绍、核心理念
- 三轮螺旋可视化展示
- 快速导航卡片
- 最新更新/推荐内容

#### 2. 课程大纲页 (Curriculum)

- 12节课时间线展示
- 每节课卡片包含：
  - 课程编号和标题
  - 学习目标
  - 核心活动
  - 交付成果
  - 相关资源链接

#### 3. 知识卡片页 (Knowledge)

- 分类导航（平面/色彩/空间）
- 卡片网格布局
- 每张卡片包含：
  - 理论要点
  - 案例缩略图
  - 难度标签（Base/Advance/Stretch）
  - 点击查看详情

#### 4. 案例库页 (Cases)

- 筛选器（专业、构成原理、难度）
- 瀑布流/网格布局
- 案例卡片包含：
  - 作品图片
  - 作品名称
  - 专业标签
  - 难度标签

#### 5. AI工具箱页 (AI Tools)

- 提示词模板库
- 工作流指南
- 常见问题解答
- 示例展示

#### 6. 学习资源页 (Resources)

- 分类列表
- 资源卡片（工具、教程、文章）
- 外部链接

## 组件设计

### 核心组件

1. **DifficultyBadge** - 难度徽章（Base/Advance/Stretch）
2. **CourseCard** - 课程卡片
3. **KnowledgeCard** - 知识卡片
4. **CaseCard** - 案例卡片
5. **ResourceCard** - 资源卡片
6. **SpiralDiagram** - 三轮螺旋图示
7. **FilterBar** - 筛选栏
8. **Navigation** - 导航栏

### 交互设计

- 卡片悬停效果
- 平滑滚动
- 筛选器动画
- 模态框展示详情
- 响应式菜单
